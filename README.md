### 项目简述
* 模拟建立一个简单基本功能的购物商城网页，实现的功能主要包括首页，详情页，登陆页，购物车页
* 项目数据来源都是京东的实时数据
* 项目主要显示动态及渲染部分，至于静态部分，书写较少

### 技术要点
* 页面主体采用主要是jquery跟js原生混合书写
* 页面数据的交互，采用的是转存locationstorage的方式

#### 项目提升空间
 * 添加更多的静态页面，丰富项目
 * 主页商品列表或者搜索关键字获取的数据总数没有定数，对于分页器的总页数规定不好操作
 * 暂时购物车里的统计是所有存在购物车页的商品，不论选中与否
 * 详情页的添加，减少暂时不能同步到购物车
